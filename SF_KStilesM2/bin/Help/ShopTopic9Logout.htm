If the users cart is empty, clicking the Logout button will log the user out and take them back to the User Logon page. If the users cart has products in it the user will be given a prompt that says "Logging out before finalizing orders will clear your cart. Do you still wish to Logout?" If the user chooses Yes, the cart will be cleared and the user will be logged out and taken abck to the User Logon page. If the user chooses No then they will stay on the Shop page.
